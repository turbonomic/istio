The turbo adapter collects Istio metrics and makes them available to Turbo.

This adapter supports the metric template.

Params

Configuration format for the turbo adapter.

Field Type Description
url string

The Turbo URL
user string

The turbonomic admin user name
password string

The turbonomic admin password
targetVersion string

The turbonomic version
adapterId string

The Istio mixer adapter id
